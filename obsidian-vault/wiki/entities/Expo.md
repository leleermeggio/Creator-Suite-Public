# Expo

React Native framework for Creator Zone frontend. Uses Expo Router (file-based routing).

## Role
- Tab navigator app under `app/(tabs)/`
- Dynamic routes: `app/project/[id].tsx`, `app/tool/[id].tsx`
- Web build via `npx expo export --platform web`
- Dev: `npx expo start --web`
